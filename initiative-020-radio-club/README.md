# Initiative-020 — Инженерный радиокружок (радио/робототехника)

Trace: VIS-001 -> VIS-010 -> VIS-011 -> VIS-040 -> STR-010
Evidence: расписание/фото демо/ссылки на проекты + список участников + ретеншн

## Source of truth
- Strategy: `docs/30-strategy.md` (STR-010)
- HOWTO: `docs/00-howto.md`

## Цель
Сделать повторяемый формат кружка, который:
- дает осязаемые инженерные результаты (демо/прототипы),
- растит локальное STEM‑сообщество (Мордовия),
- масштабируется (формат, методика, набор проектов, партнерства).

## Как работаем (коротко)
- Маленькие циклы: 4 недели пилот -> ретро -> следующий цикл.
- Каждая встреча заканчивается демо/прогрессом.
- 1–2 базовых проекта на цикл (чтобы не расползаться).

## Структура папки
- `00-admin/` — формат, аудитория, правила, безопасность, коммуникации.
- `10-sessions/` — планы встреч + чеклисты.
- `20-projects/` — проекты (описания, BOM, шаги, демо).
- `30-equipment/` — оборудование, закупки, хранение.
- `40-venue-partners/` — площадка, партнеры, договоренности.
- `50-metrics/` — метрики, ретро, evidence.
